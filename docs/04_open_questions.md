# KidsChef Open Questions

This file holds unresolved product questions that should not be treated as committed v1 scope until Product Triage records a decision.

## Current Open Questions

- Exact definition of "younger kids" for stricter stove and oven restrictions inside the broader 10-14 target range.
- Whether remote AI will be supported in v1 at all, or only documented as a supervised fallback path.
- How much local analytics is necessary for the parent dashboard without weakening privacy expectations.
- Whether favorites are device-local only or shared across devices on the same household network.
- Whether partial offline mode includes previously viewed recipes only or also ingredient-based recommendation flows.
